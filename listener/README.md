## Setup
Clone the repo:
```sh
git clone https://github.com/gbenson/monero-node.git
cd monero-node/listener
```
Create a virtual environment:
```sh
python3 -m venv venv
. venv/bin/activate
```
Upgrade pip and setuptools:
```sh
pip install --upgrade pip setuptools
```
Install in editable mode:
```sh
pip install -e .[dev]
```
